# Phase 4, Plan 3: Cursor, Cline, and Windsurf Adapters

## Goal
Implement adapters and config-injection logic for IDE-based hosts using workspace or user-specific settings files.

## Requirements Mapped
- PLAT-03: Cursor
- PLAT-04: Cline
- PLAT-05: Windsurf

## Proposed Changes
1. **`src/adapters/ide/cursor.ts`**
   - Implement adapter.
   - Add installer logic to locate and modify `.cursor/user/settings.json` or equivalent global config paths.

2. **`src/adapters/ide/windsurf.ts`**
   - Implement adapter.
   - Add installer logic to locate and modify Windsurf's configuration.

3. **`src/adapters/ide/cline.ts`**
   - Implement adapter.
   - Add installer logic for Cline's configuration.

4. **`src/adapters/ide/installer.ts`**
   - Shared utility to safely parse, modify, and format JSON/JSONC configuration files for IDEs, avoiding destructive overwrites.

## Verification
- Unit tests verifying safe JSON parsing, injection, and extraction of MCP block in mocked IDE config strings.
