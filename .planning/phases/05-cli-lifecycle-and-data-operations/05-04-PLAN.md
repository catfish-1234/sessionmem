<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user path arg → filesystem | export writes / import reads a user-supplied path (V12) |
| import JSON → service | external file content reaches importMemories (V5) |
| CLI → memories.db delete | forget --force deletes a row |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-10 | Tampering | export [path] / import <path> | mitigate | Resolve user paths with path.resolve; read/write only the explicit path; never interpolate into a shell (V12) |
| T-05-11 | Tampering / DoS | malformed or malicious import JSON | mitigate | JSON.parse wrapped with a clear stderr error; every record validated by importMemoryRecordSchema in importMemories; invalid file rejected with non-zero exit (V5) |
| T-05-12 | Denial of Service | accidental destructive forget | mitigate | D-09 dry-run-by-default: bare forget previews and never deletes; deletion requires explicit --force |
| T-05-13 | Tampering | import overwriting existing records | mitigate | Default skips duplicate IDs (non-destructive); overwrite only with explicit --merge (D-12) |
| T-05-09 | Information Disclosure | export plaintext JSON of stored content | accept | Exported memories contain whatever was summarized; redaction is SECU-02 (Phase 6) out of scope; note for docs (QLTY-03) |
</threat_model>
