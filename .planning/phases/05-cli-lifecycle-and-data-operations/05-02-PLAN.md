## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-04 | Tampering | uninstall --purge DB deletion | mitigate | --purge is explicit opt-in; default uninstall preserves memories.db (D-06); delete scoped to memories.db only, never the ~/.sessionmem dir or logs (Open Q3) |
| T-05-05 | Denial of Service | accidental destructive uninstall | mitigate | DB-preserving default means a bare `uninstall` cannot lose memories; purge requires the flag |
| T-05-06 | Repudiation | install partial-failure visibility | mitigate | ✗ step + actionable instructions printed to the user; non-zero exit (D-03/D-05) so failure is observable and scriptable |
| T-05-02 | Information Disclosure | adapter config edits in user home | accept | Adapter install/uninstall already own JSONC-aware edits; runs as invoking user (ASVS L1) |
</threat_model>
